【Webデザインの基礎】

Webサイトの改善サイクル(PDCA)

HCD（Human-CenteredDesign）/人間中心設計
・便利に使えるかどうか
・ユーザビリティの向上（使いやすさの度合い／理解しやすく、また操作がしやすいかどうか）
・アクセシビリティの向上（どんな人でも使えるWebサイトかどうか／身体的、年齢的条件に関わらず、情報を享受できる）
・長く使ってもらえるかどうか

HCDの基本フレームワーク
・UI（Webサイトの見た目の設計）
・情報設計・IA（ユーザが検索しやすく掲載情報を整理すること）
・アクセシビリティ
・ユーザビリティ

■UIデザイン
Interface：操作画面や操作方法のこと（ユーザの設置面）
UIは画面遷移も含まれる

■UXデザイン
UXを改善するためにUI・IAを見直す

■色彩
